1)добавление продукта в чек-лист
адрес:/product/add
параметры:
id_productname: integer-уникальный идентификатор названия продукта из справочника названий продуктов
id_list: integer-уникальный идентификатор списка продуктов пользователя
quantity: integer-необязательный параметр(по умолчанию-1) количество одинаковых продуктов
2)изменение продукта
адрес:/product/update
параметры:
id_product: integer-уникальный идентификатор редактируемого продукта
id_productname: integer-уникальный идентификатор названия продукта из справочника названий продуктов
id_list: integer-уникальный идентификатор списка продуктов пользователя
quantity: integer-необязательный параметр(по умолчанию-1) количество одинаковых продуктов
id_status: integer-уникальный идентификатор статуса из справочника статусов продуктов
3)вывод чек-листа пользователя
адрес:/user/list
параметры:
id_user: integer-уникальный идентификатор пользователя
id_list: integer-уникальный идентификатор списка продуктов пользователя
4)добавление пользователя
адрес:/user/add
параметры:
email: string-почта пользователя
name: string-имя пользователя
nickname: string-никнейм
password: string-пароль
5)для изменения пользователя нужно:
адрес:/user/update
параметры:
email: string-почта пользователя
name: string-имя пользователя
nickname: string-никнейм
password: string-пароль
id_user: integer-уникальный идентификатор изменяемого пользователя
6)добавление чек-листа
адрес:/list/add
параметры:
id_user: integer-уникальный идентификатор пользователя
7)вывод справочника названий продуктов
адрес:/product/namesall
параметры не задействуются
8)удаление продукта
адрес:/product/delete
параметры:
id_product: integer-уникальный идентификатор удаляемого продукта
9)распознавание продукта по фотографии
адрес:/product/scan
параметры:
id_user: integer-уникальный идентификатор пользователя
id_list: integer-уникальный идентификатор списка продуктов пользователя
