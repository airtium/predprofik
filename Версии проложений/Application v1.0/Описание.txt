В этой версии приложения, сделан список продуктов и рабочая кнопка 'add'. Для реализации списка использовался RecyclerView,
как и требовалось в т.з. Так же реализован класс для хранения словаря состоящего из названий продуктов - ключей и их id - значений,
это нужно для того, чтобы в будующем было легко делать запросы и получать ответы от сервера с базой данных только так как не подключена
база данных к приложению, то каждый раз при заходе пользователя списоксбрасывается. В методе onCreate инициализируются сам список и Adapter, 
который нужен чтобы наполнять сам список названиями продуктов. Так же инициализируется кнопка 'add', по нажатию на нее открывается диалоговое
окно, в котором есть поле для ввода названия продукта и кнопка 'save'. При нажатии на кнопку закрывается диалоговое окно и то название, которое 
написанно в поле добавляется в список. Кнопки закрыть на диалоговом окне нет, его можно закрыть нажав в свободное место на экране.
