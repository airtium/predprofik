В этой версии уже выполнены почти все начальные задачи. Теперь при нажати на кнопку scan появляется экран,на котором виден предпросмотр фото
и есть 2 кнопки: cancel, recognize. Пока работает только первая. Чтобы такой экран работал, было решено создавать новое Activity под названием
CameraActivity. И уже в нем происходит проверка включены ли у пользователя нужные разрешения и подключение к камере телефона.
Так же в этой версии приложения было реализовано подключение к БД. Для того, чтобы собирать данные в  папке проекта есть файлы, которые 
содержат модель запроса, которую сервер возвращает или запрашивает. Чтобы создавать запросы были созданы файлы, в которых происходит инициализация
доменного имени и самих запросов. В файле с именем Repository программа передает в api нужные параметры, а во ViewMdodel'и  запрос наполняется данными,
которые ввел пользователь.
